# Initialisation/seed de la base de données

Pour initialiser la base de données avec quelques données de test :

- Créez un utilisateur manuellement ou via l’API signup
- Ajoutez quelques animaux via l’API ou en base
- Ajoutez des favoris via l’API

> Il n’existe pas de script de seed automatique fourni pour éviter tout risque sur Neon. Utilisez les routes API ou un outil comme DBeaver/pgAdmin pour insérer des données de test si besoin.
