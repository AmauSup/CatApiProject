# Support de soutenance – Cat API Project

## Présentation du projet
- Application fullstack autour de The Cat API
- Stack : React (Vite), Node.js, Express, PostgreSQL, JWT

## Démonstration
- Authentification (inscription, connexion)
- Recherche et affichage de chats
- Gestion des favoris
- Tournoi de votes

## Architecture
- Frontend React (SPA)
- Backend Express (API REST, JWT)
- Base PostgreSQL
- Voir [architecture.md](architecture.md)

## Points forts
- Séparation claire frontend/backend
- Authentification sécurisée
- Documentation complète (OpenAPI, DBML, PlantUML, ADR)

## Retours d'expérience
- Difficultés rencontrées (CORS, DB, synchronisation)
- Améliorations possibles (tests, CI/CD, rôles, UI/UX)
